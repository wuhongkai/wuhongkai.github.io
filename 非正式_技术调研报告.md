# 个人作品集网站技术调研报告

## 需求概述
- 目标：搭建个人作品集网站
- 约束：使用免费工具，可接受手动上传
- 初步方案：AI coding平台 + 静态页面 + Bilibili视频 + GitHub博客

---

## 方案一：GitHub Pages + 静态网站生成器

### 核心组件
1. **托管平台：GitHub Pages**
   - 完全免费
   - 自动部署（push即发布）
   - 支持自定义域名
   - 限制：仓库1GB，月流量100GB

2. **静态网站生成器选择**
   - **Hugo**：速度最快，Go语言编写
   - **Jekyll**：GitHub原生支持，Ruby编写
   - **Astro**：现代化，支持多框架组件
   - **VuePress/VitePress**：适合技术文档风格

3. **视频存储：Bilibili**
   - 免费无限存储
   - 获取嵌入代码插入网页
   - 备选：YouTube（需翻墙）

### 优势
- 零成本
- 版本控制
- 自动化部署
- SEO友好

### 劣势
- 仅支持静态内容
- 需要学习Git基础

---

## 方案二：Vercel/Netlify + 现代前端框架

### 核心组件
1. **托管平台**
   - **Vercel**：Next.js官方，部署极快
   - **Netlify**：功能丰富，表单处理
   - 两者都提供免费套餐，自动HTTPS

2. **前端框架**
   - **Next.js**：React生态，SSG支持
   - **Nuxt**：Vue生态
   - **纯HTML/CSS/JS**：最简单

3. **AI Coding平台**
   - **v0.dev**：Vercel出品，AI生成组件
   - **Bolt.new**：StackBlitz出品，全栈开发
   - **Cursor/Claude Code**：本地开发辅助

### 优势
- 更强大的功能（表单、API）
- 现代化UI组件
- 更好的性能优化

### 劣势
- 学习曲线较陡

---

## 方案三：Notion + Super/Potion（推荐新手）

### 核心组件
1. **内容管理：Notion**
   - 可视化编辑
   - 无需代码
   - 免费个人版

2. **网站生成**
   - **Super.so**：$12/月（有免费试用）
   - **Potion**：$8/月起
   - **Notion原生分享**：完全免费但功能受限

### 优势
- 零代码
- 快速上线
- 易于维护

### 劣势
- 定制化受限
- 部分服务收费

---

## 推荐方案组合

### 方案A：技术向（推荐）
```
GitHub Pages + Hugo/Astro
+ Bilibili视频嵌入
+ GitHub Issues作为博客评论
+ Cloudflare CDN加速（可选）
```

**实施步骤：**
1. 选择Hugo主题（如PaperMod、Stack）
2. 本地编写Markdown内容
3. 推送到GitHub自动部署
4. Bilibili上传视频获取嵌入代码

**成本：** 完全免费
**时间：** 1-2天上线

### 方案B：快速启动
```
v0.dev生成页面
+ Vercel部署
+ Bilibili视频
```

**实施步骤：**
1. 用v0.dev生成作品集模板
2. 手动调整内容
3. 连接GitHub仓库到Vercel

**成本：** 免费
**时间：** 半天上线

## 视频存储方案对比

| 平台 | 优势 | 劣势 |
|------|------|------|
| Bilibili | 国内访问快、无限存储 | 需审核、有水印 |
| YouTube | 国际化、专业 | 国内访问慢 |
| 阿里云OSS | 稳定、可控 | 有成本（约0.12元/GB/月）|
| GitHub LFS | 集成方便 | 限制1GB免费 |
